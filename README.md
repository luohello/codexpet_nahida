# codexpet_nahida

A Codex v2 animated pet package for Nahida.

## Files

- `pet.json`: Codex pet metadata.
- `spritesheet.webp`: v2 animated spritesheet.

## Install

Copy this folder to your Codex pets directory:

```powershell
Copy-Item -Recurse -Force . "$env:USERPROFILE\.codex\pets\nahida"
```

Then select `nahida` as the active pet in Codex.

## Character

Nahida is cute but dependable, and she directly points out problems when she notices them.
